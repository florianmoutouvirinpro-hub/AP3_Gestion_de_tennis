AP3 — Conformité RGPD et gestion des rôles — Club de Tennis (2 éme année): Extension de l'application Club de Tennis avec mise en conformité RGPD : 3 rôles implémentés (Administrateur, Lecteur, DPO) avec permissions granulaires, trigger SQL BEFORE DELETE archivant les données supprimées dans une table HISTORIQUE (conservation 3 ans). Authentification Bcrypt, conformité articles 5, 13-14, 17, 32 et 37 du RGPD. Documents : MCD/MPD, script SQL (ROLE + HISTORIQUE + trigger), code source PHP, documentation conformité RGPD.
AP2 — Gestion du personnel et du bureau — Club de Tennis (2 éme année): Extension d'une application web de gestion d'un club de tennis pour intégrer la gestion du personnel (moniteurs, stagiaires, secrétaires) et les mandats datés des membres du bureau. CRUD complet, contrôle de l'unicité du président, validation des dates, sécurité PDO et protection XSS. Technologies : PHP, HTML, CSS, MySQL, PhpMyAdmin. Documents : Cahier des charges, MCD, script SQL, code source PHP, documentation synthétique.
